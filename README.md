# CCL MiFi Tracker

Colleton County Library — MiFi Device Checkout Tracking System

## Live App
**https://lrdsatyr8.github.io/mifi-tracker**

## What It Does
- Tracks all MiFi devices across COL-HQ, COL-COT, COL-ED, and COL-BK
- Check devices in and out to patrons with automatic 21-day due dates
- Flags overdue devices and updates patron late counts automatically
- Add, edit, or delete any device or patron record
- Full transaction log of every checkout, return, edit, and deletion
- All data stored permanently in Supabase cloud database

## Access
Open the live app URL in any browser on any device — phone, tablet, or PC.
No login required. All changes save instantly to the cloud.
