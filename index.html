Import / Export

Export pulls live data from the database as a CSV file. Import uses upsert (add new + update existing) for devices and patrons, and appends for history logs.

📡 Devices

device_num, barcode, imei, last5, phone, status, due_date, under_contract, being_billed, library, patron_id, patron_email, notes

👤 Patrons

patron_id, email, blocked, warned, late_count, notes

📋 History / Log

id, ts, type, msg, created_at

Import notes: First row must be the header row with exact column names. Devices and Patrons use upsert - existing records with the same ID are updated, new ones are added. History import appends only rows without an id (new entries). After import, data reloads automatically.
